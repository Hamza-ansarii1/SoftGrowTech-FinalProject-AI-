# 🤖 AI Customer Support Chatbot

An AI-powered customer support chatbot built with Python. It uses Natural Language Processing (NLP) with TF-IDF Vectorization and Cosine Similarity to understand user queries and return the most relevant response.

## 🚀 Features
- Answer customer questions
- Automated responses
- NLP-based text matching
- TF-IDF Vectorizer
- Cosine Similarity
- Interactive command-line chatbot

## 🛠️ Technologies Used
- Python
- NLTK
- Scikit-learn

## 📦 Installation

```bash
pip install nltk scikit-learn
```

## 💬 Example

```text
You: hello
Bot: Hello! How can I help you?

You: payment
Bot: We accept Visa, MasterCard, JazzCash, and Easypaisa.

You: refund
Bot: Refunds are available within 7 days.

You: bye

```
